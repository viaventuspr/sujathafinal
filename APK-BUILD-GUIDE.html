⚡ Method 1: PWABuilder (සිනාසෙන තරම් ලේසි!)

Microsoft ගේ free tool. Coding කිසිවක් ඕනෑ නැහැ!

STEP 1

Deploy to Netlify (ZIP upload)

පළමුව sujatha-netlify.zip Netlify deploy කරන්න:

1. netlify.com → New Site → Deploy Manually

2. ZIP drag & drop කරන්න

3. Live URL ලැබෙනවා: https://your-site.netlify.app

GitHub Pages use කරනවා නම් https://viaventuspr.github.io/sujatha/ ඒ URL use කරන්න
STEP 2

PWABuilder වලින් APK Generate

1. pwabuilder.com visit කරන්න

2. ඔබේ Netlify URL paste කරන්න (e.g. https://your-site.netlify.app)

3. Start click කරන්න → Website analyze කරනවා

4. Package for Stores → Android → Generate Package

5. APK download වෙනවා! ✅

PWABuilder automatically manifest.json read කරලා APK build කරනවා. Icons, name, theme color — සියල්ල automatic!
STEP 3

APK Install on Android

1. APK file phone එකේ download folder එකට copy කරන්න

2. Phone Settings → Security → Unknown Sources enable කරන්න

3. APK file tap කරන්න → Install

4. Home screen එකේ app icon දිස්වෙනවා! 🎉

🔧 Method 2: Bubblewrap (Google Official)

Terminal use කරන advanced users සඳහා. Signed APK + Play Store ready!

STEP 1

Prerequisites Install

# Node.js install (https://nodejs.org) node --version # v16+ ඕනෑ # Bubblewrap install npm install -g @bubblewrap/cli # Java JDK (Android build needs Java) # Download: https://adoptium.net java --version # v11+ ඕනෑ
STEP 2

TWA Project Initialize

# New folder create mkdir sujatha-android && cd sujatha-android # Initialize with your Netlify URL bubblewrap init --manifest https://your-site.netlify.app/manifest.json # Follow prompts: # App name: Sujatha Malani # Package: lk.edu.sujathamalani # Version: 1.0.0
STEP 3

APK Build

# Build APK bubblewrap build # Output: app-release-signed.apk # This is installable on Android!
First build — Android SDK auto-download කරනවා (~1GB). Internet connection ඕනෑ!

📲 Method 3: Direct Phone Install (Quickest!)

APK build නොකර direct install කරන fastest ක්‍රමය:

EASIEST

Android Chrome → Add to Home Screen

1. Phone Chrome browser → ඔබේ website open කරන්න

2. Menu (⋮) → "Add to Home Screen"

3. Name: "Sujatha Malani" → Add

4. Done! Home screen icon දිස්වෙනවා 🎉

ඒ icon click කළ විට — standalone app ලෙස launch වෙනවා (browser UI නැහැ, full screen, splash screen show වෙනවා!)
